# 🚚 화물 혼적 경로 — AI 화물 운행 비서

여러 배송지를 입력하면 **AI가 가장 효율적인 방문 순서를 자동 계산**하고,
TMAP · 카카오맵 · 네이버지도로 바로 길안내를 시작하는 화물기사 전용 웹앱입니다.

> 📱 **휴대폰·PC 어디서나 바로 사용:** https://victore7255-source.github.io/cargo-route/
> HTTPS라서 📡 현재위치(GPS) 버튼도 동작합니다. 휴대폰 브라우저에서 열고 **홈 화면에 추가**하면 앱처럼 쓸 수 있어요.

**API 키·회원가입·설치 없이** 바로 동작합니다.
(지오코딩: OpenStreetMap Nominatim, 경로 최적화: OSRM 공개 서버 — 모두 무료)

- 🎨 밝은 그린 톤의 기본 테마 + 야간 운행용 **다크모드** (오른쪽 위 🌙 버튼, 시스템 설정 자동 감지)
- 👀 **미리보기**: 주소 입력 없이 화면을 구경하려면 `http://localhost:8787/?demo` 로 열기 (예시 데이터, 저장 안 됨)
- 📸 화면 스크린샷: `docs/스크린샷/` 폴더

---

## ▶️ 실행 방법

### 방법 1 — 더블클릭 (추천)

1. 폴더 안의 **`실행하기.command`** 파일을 더블클릭
2. 브라우저가 자동으로 열립니다 (`http://localhost:8787`)
3. 처음 실행 시 macOS가 차단하면: 파일에서 **마우스 오른쪽 클릭 → 열기 → 열기**

> 이 방법으로 열어야 **📡 현재위치(GPS)** 버튼이 동작합니다.

### 방법 2 — 터미널

```bash
cd ~/Downloads/화물혼적경로
python3 -m http.server 8787
```
그다음 브라우저에서 `http://localhost:8787` 접속.

### 휴대폰에서 쓰려면

GitHub Pages에 배포되어 있습니다 — 휴대폰에서 아래 주소를 열면 됩니다.

**https://victore7255-source.github.io/cargo-route/**

내비게이션 연동(TMAP/카카오맵/네이버 버튼)은 휴대폰에서 열어야 앱이 실행됩니다.
**홈 화면에 추가**하면 앱처럼 사용할 수 있습니다.

코드를 수정한 뒤 다시 배포하려면:

```bash
cd ~/Downloads/화물혼적경로
git add -A && git commit -m "수정 내용" && git push
```
(1~2분 뒤 위 주소에 자동 반영됩니다)

## 📱 사용 순서

1. **경로 탭** — 출발지 입력(또는 📡 현재위치) → 배송지를 한 줄에 하나씩 붙여넣고 **＋ 배송지 추가**
2. 각 배송지의 **하차/상차** 배지를 눌러 작업 종류 변경, 필요하면 **최종 목적지** 지정
3. **⚡ AI 최적 경로 계산** → 방문 순서 · 총거리 · 예상시간 · 통행료 · 도착 예정시각 · 지도 · AI 운행 추천
4. **TMAP / 카카오맵 / 네이버지도** 버튼으로 첫 목적지 안내 시작
5. **🚛 운행 시작** → 운행 탭에서 목적지마다 [도착] · [상차/하차 완료] 체크
6. **🏁 운행 종료** → 통행료·충전비 입력 후 저장 → **기록 탭**에서 월별 통계 확인
7. **적재 탭** — 박스 크기·수량 입력 → CBM · 적재율 · 최대 적재 수 · 추천 적재 방법
8. **EV 탭** — 계기판의 주행가능거리(km) 입력 → 완주 가능 여부 · 충전 필요 지점 · 충전소 검색

## ⚠️ 참고

- **통행료·소요시간은 추정치**입니다. 정확한 통행료는 실제 서비스 단계에서 TMAP/카카오모빌리티 API 연동이 필요합니다(기획서 참고).
- 무료 공개 서버를 사용하므로 주소 확인이 건당 약 1초씩 걸립니다(서버 이용 정책).
- 모든 데이터(배송지·운행기록·차량 설정)는 **내 브라우저에만 저장**되며 외부로 전송되지 않습니다.
